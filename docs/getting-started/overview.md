# ThreatGuardとは

> **ひとことで：** 自社ブランドになりすました偽サイトを見つけて、削除依頼まで行えるサービスです。

ThreatGuard は、フィッシングサイトやブランドなりすましドメインの **検知・調査・削除申請** をワンストップで提供するクラウドプラットフォームです。

## 解決する課題

金融機関・暗号資産交換業者・ECサービスなどは、自社ブランドを悪用したフィッシングサイトの被害に常にさらされています。

| 課題 | 従来の対応 | ThreatGuard |
|------|-----------|-------------|
| 偽サイトの発見 | 顧客からの通報頼み | 自動スキャンで能動的に検出（ドメイン追加時にも自動実行） |
| 脅威の判定 | セキュリティ担当が手動調査 | AIが自動分類・スコアリング |
| 削除依頼 | 英語のメールを手書き | AI文面生成・自動送信 |
| 進捗管理 | Excelやメールで属人管理 | ダッシュボードで一元管理 |
| 報告 | 都度手作業で作成 | レポート自動エクスポート |

## 想定ユーザー

ThreatGuard は以下の2つのユーザー像を想定しています。

### 法務・AML部門の方

- フィッシングやDNSの専門知識がなくても使えるよう設計されています
- 平易な日本語でのガイダンスと、「次に何をすべきか」の明確な導線
- 調査結果はそのまま上層部・当局への報告に使える品質

### システムセキュリティ部門（CISO）の方

- DNS解決結果、HTTPヘッダー、SSL証明書、WHOIS情報などの生データを確認できます
- リスクスコアリングのロジック（重み付け・算出方法）を透明に公開
- APIエンドポイント仕様も提供予定

## 主要機能の全体像

ThreatGuard は以下の流れでブランド保護を実行します。

```
ブランド登録（ロゴ・商標証明アップロード・ドメイン2層管理対応）
    ↓ セットアップチェックリスト（4ステップガイド）
    ↓
① ドメインスキャン ─── 類似ドメインの発見（登録・ドメイン追加時に自動実行）
    ↓
② ページ調査 ─────── 実際にサイトにアクセスして内容を調査
    ↓
③ リスク判定 ─────── AI分析 + 5要素スコアリング
    ↓
④ 削除申請 ─────── 文面生成 → abuse連絡先に送信 → 進捗追跡
    │                  ＋ ブラウザ削除申請（Google Safe Browsing / SmartScreen）
    ↓
⑤ レポート ─────── 行政・取締役会・法務向けレポート自動生成
```

さらに、**SNS監視**（Twitterフィッシング拡散監視）や**Slack通知**でリアルタイムにチームへアラートを配信できます。

各機能の詳細は [機能詳細](/features/domain-scan) セクションをご覧ください。

## 対象業界

- 暗号資産交換業者
- 証券会社・銀行などの金融機関
- ECサイト・オンラインサービス
- ブランドを持つすべての企業
