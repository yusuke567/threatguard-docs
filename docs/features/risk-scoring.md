# リスクスコアリング

> **ひとことで：** 検出された脅威に0〜100のスコアを付けて、どれから優先的に対応すべきかを判断できるようにします。

リスクスコアは、複数の分析結果を重み付けして自動算出されます。スコアが高いほど、なりすましやフィッシングの可能性が高いことを示します。

## スコアの5要素

| 要素 | 重み | 概要 |
|------|------|------|
| ドメイン類似度 | **30%** | 正規ドメインとの文字列の似ている度合い |
| ドメイン年齢 | **20%** | そのドメインがどのくらい前に検出されたか |
| SSL情報 | **15%** | SSL証明書の発行元の信頼性 |
| 脅威カテゴリ（AI分析） | **25%** | AIが判定した脅威の種類 |
| コンテンツ分析 | **10%** | ページ調査で検出されたフィッシング兆候 |

### 1. ドメイン類似度（30%）

正規ドメインと検出ドメインの文字列をレーベンシュタイン距離で比較します。

| 類似度 | スコア | 例 |
|--------|--------|-----|
| 95%以上（ほぼ同じ） | 95 | `example-brand.com`（1文字違い） |
| 80〜95% | 80〜95 | `examp1e-brand.com`（ホモグリフ） |
| 60〜80% | 50〜80 | `example-brand-jp.com` |
| 40〜60% | 20〜50 | `mycheck-coin.com` |
| 40%未満 | 0〜20 | 全く別のドメイン |

::: info レーベンシュタイン距離とは
2つの文字列を一方からもう一方に変換するために必要な「挿入・削除・置換」の最小回数。少ないほど似ている。
:::

### 2. ドメイン年齢（20%）

初回検出からの経過日数で判定します。新しく作られたドメインほどリスクが高い傾向があります。

| 経過日数 | スコア | 理由 |
|---------|--------|------|
| 1日未満 | 100 | 直近に作成された（攻撃準備中の可能性） |
| 1〜7日 | 90 | 非常に新しい |
| 7〜30日 | 70 | まだ新しい |
| 30〜90日 | 50 | やや新しい |
| 90日〜1年 | 30 | 中期的に存在 |
| 1年以上 | 10 | 長期間存在（正規サイトの可能性も） |

### 3. SSL情報（15%）

SSL証明書の発行元（認証局）でリスクを判定します。

| 発行元 | スコア | 理由 |
|--------|--------|------|
| 自己署名証明書 | 90 | 正規サービスでは通常使わない |
| 情報なし | 70 | SSL未設定は不審 |
| Let's Encrypt | 60 | 無料で取得可能。フィッシングでの利用が多い |
| 有名CA（DigiCert等） | 20 | 審査を経ているため信頼性が高い |

### 4. 脅威カテゴリ — AI分析（25%）

Claude（AI）がドメイン情報・WHOIS・SSL情報を分析し、5つのカテゴリに分類します。

| カテゴリ | スコア | 説明 |
|---------|--------|------|
| **phishing** | 100 | フィッシング（認証情報の窃取を目的としたサイト） |
| **brand_abuse** | 75 | ブランドの不正使用（フィッシングではないが商標侵害） |
| **unknown** | 50 | 判定不能（情報不足） |
| **parked** | 40 | パークドメイン（登録済みだが未使用、または「売り出し中」） |
| **legitimate** | 5 | 正規サイト（ブランドに正式に関連） |

::: info AI分析の仕組み（CISO向け）
- モデル: 大規模言語モデル（LLM）
- 入力: ブランド情報（名称・ドメイン・キーワード）+ 検出ドメイン情報（ドメイン名・初回検出日・WHOIS・SSL）
- 出力: カテゴリ（5種）+ 信頼度（0〜1）+ 判定理由（テキスト）
- 判定理由は脅威詳細画面で閲覧可能
:::

### 5. コンテンツ分析（10%）

[ページ調査](/features/page-investigation) で検出されたフィッシング兆候から算出：

| 兆候 | 加点 |
|------|------|
| ブランドキーワードがHTMLに含まれる | 最大40pt（1キーワードにつき15pt） |
| ログインフォームが存在 | 25pt |
| パスワードフィールドが存在（フォームなし） | 20pt |
| ロゴ画像が検出 | 10pt |
| 画像類似度（公式サイトとの比較） | 最大25pt（類似度 × 25） |

## ブースト加算

基本スコアに加え、以下の場合にボーナス加点されます。

| 条件 | 加点 |
|------|------|
| ユーザーからの通報あり | +15pt |
| 被害者報告あり（victimCount > 0） | +10pt |

最終スコアは **0〜100の範囲** にクリップされます。

## スコアの活用

| スコア帯 | 推奨アクション |
|---------|--------------|
| **80〜100** | 🔴 即座に削除申請を検討 |
| **50〜79** | 🟡 ページ調査を実施し、状況を確認 |
| **20〜49** | 監視を継続 |
| **0〜19** | 🟢 リスク低。誤検知の可能性 |
