# 無料診断ツール

> **ひとことで：** アカウント登録なしで、気になるURLがフィッシングサイトかどうかを簡易診断できる公開ツールです。

ThreatGuard の無料診断ツールは、誰でもすぐに利用できるフィッシングサイト判定サービスです。ユーザー登録や料金は一切不要です。

## 利用方法

1. `/diagnose` ページにアクセス
2. 診断したいURLを入力
3. メールアドレスを入力（結果通知用）
4. 「診断する」をクリック

## 診断内容

入力されたURLに対して、以下の調査を自動実行します：

| 調査項目 | 内容 |
|---------|------|
| **DNS解決** | ドメインが実在するかの確認 |
| **HTTP接続** | サイトへのアクセス可否、ステータスコード |
| **リダイレクト先** | 最終的な遷移先URL |
| **IPアドレス** | ホスティング先の特定 |
| **SSL証明書情報** | 証明書の発行者・有効期限 |
| **WHOIS情報** | ドメインの登録者・登録日 |
| **スクリーンショット** | サイトの見た目をキャプチャ |
| **HTMLスニペット** | ページ内容の一部を取得 |
| **AI脅威分析** | Claude AIによるカテゴリ判定 |

## 診断結果

診断が完了すると、以下の情報が表示されます：

- **リスクスコア** — 0〜100の数値
- **脅威カテゴリ** — phishing / brand_abuse / parked / legitimate / unknown
- **信頼度** — AI判定の確信度（0〜1）
- **判定理由** — AIによる日本語の解説文
- **スクリーンショット** — サイトのキャプチャ画像

## ステータス

| ステータス | 説明 |
|-----------|------|
| **pending** | 受付完了、診断待ち |
| **scanning** | 診断実行中 |
| **completed** | 診断完了 |
| **failed** | 診断失敗（サイトにアクセスできない等） |

## 制限事項

- 診断結果の保持期間は **7日間** です。期限を過ぎると自動削除されます。
- 有料プランで提供される詳細な機能（削除申請、継続監視、レポートなど）は含まれません。
- 1回の診断で1つのURLのみ対応です。

::: tip ThreatGuard の全機能を試すには
無料診断で脅威が確認された場合、ThreatGuard のアカウントを作成いただくことで、削除申請や継続的な監視が可能になります。

:::
