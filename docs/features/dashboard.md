# ダッシュボード

> **ひとことで：** 検出された脅威の全体像を一目で把握し、対応が必要なものから順番に確認できる画面です。

## 画面構成

### サマリーカード

画面上部に3枚のカードで現在の状況を表示します。

| カード | 色 | 表示内容 |
|--------|-----|---------|
| 🔴 **対応が必要** | 赤 | 脅威が確認され、削除申請などのアクションが必要な件数 |
| 🟡 **確認待ち** | 黄 | 新規検出されたが、まだ調査・判定が完了していない件数 |
| 🟢 **対応済み** | 緑 | 削除申請が完了した、または安全と判定された件数 |

各カードをクリックすると、該当ステータスで脅威一覧がフィルタリングされます。

### 脅威一覧テーブル

検出された全ドメインを一覧表示します。

| 列 | 内容 |
|----|------|
| ドメイン名 | 検出されたドメイン |
| ブランド | 対象ブランド名（複数ブランド管理時） |
| リスクスコア | 0〜100のスコア（色分け表示） |
| ステータス | new_domain / confirmed_threat / takedown_sent / resolved 等 |
| 検出日 | 最初に検出された日時 |
| チェックボックス | 一括削除申請の対象選択用 |

- **デフォルトソート**: リスクスコア降順（高リスクが上）
- **ブランドフィルター**: ドロップダウンで特定ブランドに絞り込み
- **一括削除申請**: チェックボックスで複数の脅威を選択し、一括で削除申請ウィザードに進めます

### レイヤー2展開パネル

テーブルの各行を展開すると、詳細情報を確認できます：
- AI分析の判定理由
- 脅威カテゴリ
- 最終調査日時

::: tip 操作のコツ
まず 🔴「対応が必要」をクリックして高リスクの脅威に絞り込み、上から順に詳細を確認→削除申請に進むのが効率的です。レジストラへの削除申請と**ブラウザ削除申請**（Google Safe Browsing / SmartScreen）を併用すると、より早く被害を軽減できます。
:::

## ブランド詳細ページ

サイドバーまたはブランド名リンクからブランド詳細ページにアクセスできます。

| セクション | 内容 |
|-----------|------|
| セットアップチェックリスト | ロゴ設定・ドメイン登録・メール設定・通知設定の4ステップガイド |
| ブランド情報 | ブランド名、ロゴ画像、商標登録証明、プライマリドメイン、保有ドメイン |
| 監視ステータス | 現在の監視状態（スキャン中・待機中等）をリアルタイム表示 |
| 脅威統計サマリー | 検出脅威数、リスク分布、ステータス別件数をグラフ表示 |
| スキャン履歴 | 過去のスキャン実行履歴をページネーション付きで閲覧 |
| ドメイン管理 | プライマリドメイン・保有ドメインの追加・編集（一括追加対応） |
| メール送信設定 | テイクダウンメールの送信元設定 |

- ロゴ画像はブランド詳細ページからアップロード・変更できます（常時表示のロゴ設定ボタン）
- **商標登録証明** のアップロードにも対応。テイクダウン申請時の法的根拠を強化できます
- 保有ドメインの **一括追加** に対応。複数ドメインをまとめて登録できます
- 保有ドメインを追加すると、既存の検出脅威が自動的に **false_positive** に再分類されます

## スクリーンショット比較

脅威詳細ページでは、不審サイトと公式サイトのスクリーンショットを**並べて比較**できます。

- 不審サイトのキャプチャと公式サイトのリファレンス画像を横並び表示
- 類似度スコアを併記
- 過去の調査結果との比較も可能

## ステータス変更履歴

脅威詳細ページに、ステータスの変更履歴が時系列で表示されます。

```
new_domain → confirmed_threat → takedown_sent → resolved
```

各変更のタイムスタンプが記録され、対応の経緯を追跡できます。

## ダークモード

画面右上のテーマ切替ボタンから、3段階のテーマを選択できます。

| モード | 説明 |
|--------|------|
| ☀️ ライト | 明るい背景（デフォルト） |
| 🖥️ システム | OS設定に追従 |
| 🌙 ダーク | 暗い背景（目に優しい） |

## オンボーディング

初回ログイン時に、4ステップのオンボーディングガイドが表示されます。各ステップはクリッカブルリンクになっており、該当する設定画面に直接ジャンプできます。

1. ブランド登録
2. ドメイン設定
3. 通知設定
4. 初回スキャン確認

ThreatGuard の用語や操作に慣れるまでの参考としてご利用ください。

## SNS監視

サイドバーの「SNS監視」からTwitterフィッシングURL拡散監視画面にアクセスできます。

- 検出されたフィッシングURLの拡散状況を一覧表示
- **0件の場合**は、SNS監視の仕組みを説明する3ステップガイドUIが表示され、機能の理解を助けます
