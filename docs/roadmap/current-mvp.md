# MVP機能一覧

> 現在提供中の機能です。

## 提供中の機能

### 🔍 脅威検知

| 機能 | 状態 | 説明 |
|------|------|------|
| 類似ドメイン生成 | ✅ 提供中 | ホモグリフ・タイポ・TLD差し替え・ハイフン挿入で候補生成 |
| DNS検証 | ✅ 提供中 | 生成候補のDNS解決で実在確認 |
| CT（証明書透過性）監視 | ✅ 提供中 | SSL証明書発行ログを監視（リトライ + CertSpotterフォールバック付き） |
| 定期スキャン | ✅ 提供中 | cronスケジュールで自動実行 |
| ドメイン追加時の自動スキャン | ✅ 提供中 | ブランド作成・プライマリドメイン追加時にフルスキャンを自動実行 |

### 🌐 ページ調査

| 機能 | 状態 | 説明 |
|------|------|------|
| ヘッドレスブラウザ訪問 | ✅ 提供中 | ヘッドレスブラウザでHTTPS/HTTPアクセス |
| 画面キャプチャ | ✅ 提供中 | 1280×720 PNGスクリーンショット |
| コンテンツ分析 | ✅ 提供中 | フォーム検知・キーワードマッチ・ロゴ検出 |
| 画像類似度判定 | ✅ 提供中 | pHashによる公式サイトとの視覚比較 |
| 再調査 | ✅ 提供中 | 任意のタイミングで再実行可能 |

### 📊 分析・判定

| 機能 | 状態 | 説明 |
|------|------|------|
| AI脅威分類 | ✅ 提供中 | Claude による5カテゴリ自動分類 |
| リスクスコアリング | ✅ 提供中 | 5要素の重み付けスコア（0〜100） |
| ユーザー通報ブースト | ✅ 提供中 | 通報・被害報告によるスコア加算 |

### 📧 削除申請

| 機能 | 状態 | 説明 |
|------|------|------|
| abuse連絡先自動取得 | ✅ 提供中 | WHOIS + RDAPフォールバック |
| AI文面生成 | ✅ 提供中 | 日本語/英語の自動切替 |
| 一括送信 | ✅ 提供中 | 複数ドメインの同時申請 |
| グルーピング | ✅ 提供中 | 同一abuse宛てを1通にまとめ |
| 進捗トラッキング | ✅ 提供中 | ステータス管理（送信→回答待ち→完了/却下） |
| 再送信 | ✅ 提供中 | 却下時の文面修正→再送信 |
| Google Safe Browsing報告 | ✅ 提供中 | フィッシングサイトをGoogleに報告し、Chrome等のブラウザでフィッシング警告を表示 |
| Microsoft SmartScreen報告 | ✅ 提供中 | フィッシングサイトをMicrosoftに報告し、Edgeブラウザでフィッシング警告を表示 |

| AI | LLM API |
| メール送信 | メールAPI + SMTP |
| ドメイン | threatguard.jp |
