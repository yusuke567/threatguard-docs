## 2026-03-19 — ドキュメント v3 アップデート

3月14日〜3月19日のプロダクト変更を反映し、全ページを最新化しました。

### 🆕 新機能

| 機能 | 説明 |
|------|------|
| **招待制ユーザー登録** | 公開登録を廃止し、管理者による招待制に変更。セキュリティ強化 |
| **商標登録証明アップロード** | ブランド詳細ページから商標登録を証明する書類をアップロード可能に。テイクダウン申請の法的根拠を強化 |
| **セットアップチェックリスト** | ブランド詳細ページに4ステップの初期設定ガイドを追加（ロゴ設定・ドメイン登録・メール設定・通知設定） |
| **監視ステータス表示** | ブランド管理画面・詳細画面でスキャン状態をリアルタイム表示 |
| **スキャン履歴** | 過去のスキャン実行履歴をページネーション付きで閲覧可能に |
| **保有ドメイン一括追加** | 複数の保有ドメインをまとめて登録可能に |
| **ドメイン追加時の自動スキャン** | ブランド作成・プライマリドメイン追加時にフルスキャンを自動実行 |
| **保有ドメイン追加時の自動再分類** | 保有ドメイン追加時に、該当ドメインの脅威を自動的にfalse_positiveに再分類 |
| **パスワード再発行** | ログイン画面から登録メールアドレス宛にパスワード再発行メールを送信 |
| **パスワード表示/非表示トグル** | ログイン画面でパスワードの表示切替が可能に |
| **SNS監視の0件時説明UI** | 検出0件時にSNS監視の仕組みを3ステップで解説するガイドUIを表示 |
| **CT log リトライ + CertSpotterフォールバック** | CTログAPIの障害時に自動リトライ、CertSpotter APIへのフォールバックを追加 |
| **永続ファイルストレージ** | `DATA_DIR`環境変数による永続ボリューム対応。デプロイ時のファイル消失を防止 |

### 🔧 改善

| 項目 | 説明 |
|------|------|
| **オンボーディング改善** | 4ステップのクリッカブルリンク形式に変更。各ステップから設定画面に直接ジャンプ |
| **ブランド詳細ページ整理** | ドメイン管理・SMTP設定をブランド詳細に集約。編集フォームを簡素化 |
| **ロゴ設定ボタン常時表示** | ブランド詳細ページでロゴ設定ボタンを常に表示するよう改善 |
| **メール送信設定セクション追加** | ブランド詳細ページにメール送信設定セクションを新設 |
| **Slack通知トグル対応** | Slack Webhook URLが設定されていてもトグルOFF時は通知をスキップ |

## 2026-03-14 — ドキュメント v2 アップデート

3月5日〜3月14日のプロダクト変更を反映し、全ページを最新化しました。

### 🆕 新機能

| 機能 | 説明 |
|------|------|
| **Twitter フィッシングURL拡散監視** | ブランドに関連するフィッシングURLのTwitter上での拡散を自動監視（Phase 1） |
| **ブランド詳細ページ** | ロゴアップロード、脅威統計サマリー、ドメイン管理を集約したブランド専用ページ |
| **ドメイン2層管理** | プライマリドメイン（監視対象）と保有ドメイン（ホワイトリスト＋期限切れ監視）の2層管理 |
| **スクリーンショット比較** | 不審サイトと公式サイトのスクリーンショットを並べて比較表示 |
| **ステータス変更履歴** | 脅威ステータスの変更を時系列で表示（対応経緯の追跡） |
| **ダークモード** | ライト / システム / ダーク の3段階テーマ切替 |
| **一括削除申請** | TOPページからチェックボックスで複数脅威を選択し、一括で削除申請ウィザードへ |
| **組織ごとのSlack通知設定** | 組織単位でSlack Webhook URLを個別設定可能 |
| **テストメール送信** | 通知設定画面から動作確認用のテストメールを送信可能 |
| **Resend APIメール送信** | テイクダウン・通知メールの送信基盤をResend APIに移行（信頼性向上） |

### 🔧 改善

| 項目 | 説明 |
|------|------|
| **「脅威確認」ステータス追加** | `confirmed_threat`（脅威確認）の日本語ラベルを追加 |
| **メール送信タイムアウト対策** | 大量メール送信時のタイムアウトエラーを改善 |
| **一括申請の空items送信防止** | 未選択時のエラーメッセージ改善 |
| **ログイン画面セキュリティ** | プレースホルダーを汎用的なものに変更 |
