# 準拠規格・法令

> **ひとことで：** ThreatGuard のテイクダウン活動に関連する国際ルールと日本の法制度をまとめます。

## テイクダウンの法的根拠

ThreatGuard が削除申請で引用する主な法的根拠です。

### 国際ルール

| ルール | 概要 | ThreatGuardでの活用 |
|--------|------|-------------------|
| **UDRP** | ICANNの統一ドメイン名紛争処理方針。商標侵害ドメインの停止・移転を求める国際的な枠組み | テイクダウンメールで引用 |
| **ICANN RAA §3.18** | レジストラ認定契約。不正利用への対応義務を規定 | テイクダウンメールで引用 |
| **APWGベストプラクティス** | Anti-Phishing Working Group のフィッシング対策ガイドライン | テイクダウンメールで引用 |

### 日本の法制度

| 法令 | 概要 | 関連する場面 |
|------|------|-------------|
| **不正競争防止法** | ブランドの模倣や混同を招く行為を規制 | フィッシングサイトによるブランド侵害の根拠 |
| **商標法** | 登録商標の不正使用を規制 | 自社商標を使用した偽サイトへの対応 |
| **不正アクセス禁止法** | 不正にIDやパスワードを取得する行為を禁止 | フィッシングによる認証情報の窃取に該当 |
| **個人情報保護法** | 個人データの適切な取り扱いを義務付け | ユーザーデータの管理 |

## セキュリティ関連の通報先

フィッシングサイトを発見した場合の通報先一覧です。ThreatGuard のテイクダウンと並行して活用できます。

| 通報先 | 対象 | URL |
|--------|------|-----|
| **JPCERT/CC** | 日本国内のインシデント全般 | https://www.jpcert.or.jp/ |
| **フィッシング対策協議会** | フィッシングサイトの報告 | https://www.antiphishing.jp/ |
| **警察庁サイバー犯罪相談窓口** | 犯罪被害の相談 | 各都道府県警察のサイバー犯罪相談窓口 |
| **金融庁** | 金融機関のセキュリティインシデント | 各管轄局の担当窓口 |
| **Google Safe Browsing** | フィッシングサイトのブラウザ警告 | https://safebrowsing.google.com/safebrowsing/report_phish/ |

## 暗号資産交換業者向けの規制

暗号資産交換業者は以下の追加規制の対象です。

| 規制 | 要点 |
|------|------|
| **資金決済法** | 暗号資産交換業の登録制度、利用者保護 |
| **犯罪収益移転防止法（犯収法）** | AML/CFT（マネーロンダリング・テロ資金対策）義務 |
| **JVCEA自主規制** | 日本暗号資産取引業協会の自主規制ルール |

フィッシング被害の防止は、これらの規制における「利用者保護」の重要な要素です。
